# Failure-Dashboard



## To run the dashboard, perform the following steps:
1. Install Docker Desktop on your laptop and have the docker engine running.
2. Run the following command in root dir: `docker-compose up -d --build`.
3. After successful run, the log dashboard is started at port 80. The url to check the testcases is: `http://localhost/dashboard/c163bbf3` (c163bbf3 = id of the testcase you want to analyse).
